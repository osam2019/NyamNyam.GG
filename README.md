# 냠냠.GG
### 부제:오늘 뭐 먹을까?
<hr/>

냠냠.GG는 **실시간 음식 순위**를 알려주는 사이트입니다.
냠냠.GG는 현재
- 네이버(카카오, 인스타그램은 추후 추가 예정) 검색결과를 바탕으로 현재 인기있는 음식 알려주기
- 내 근처에 먹고 싶은 음식을 잘하는 음식점이 어디있는지 알려주기

이 서비스를 제공하고 있습니다.
